This plugin add a search bar above your products category list. If you are running an online store, you need this!

== Description ==

If your site has a large number of product categories, it can be helpful to be able to search through them rather than scrolling to find the one you need.
This plugin adds a text box to the category list which filters the list in place.

== Installation ==

To install this plugin just follow the standard procedure.

or

1. Upload `font-awesome-integration.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
